Confirmed Scaling Formulas

Stats(N) = 1 + 0.01 * N (linear, confirmed) RL(N) = 1 + 0.01 * N (linear, confirmed) RB(N) = 1 + 0.01 * N (linear, confirmed) Clone(N) = 0.05% * N (linear, confirmed) Cost scaling: 1, 13, 23, 50 (non-linear, roughly quadratic) Success: 100%, 59%, 44%, 23% (decreasing ~2-3% per level) Expected shards to reach level N (accounting for failure): E[shards for level k] = Cost(k) / SuccessRate(k) E[total to N] = sum(k=1 to N) of Cost(k)/Success(k)

Gacha Expected Cost Calculator

Stats(N) = 1 + 0.01 * N Clone(N) = 0.05% * N Cost(N) ~ quadratic: Cost(k) ~ 0.06*k^2 (rough fit) Success(N) ~ 100% - 2.75%*N (rough linear fit) Expected shards to reach level N: E[total] = SUM(k=1..N) Cost(k) / Success(k) Example: Level 28 Stats boost: x1.28 (all three) Clone: +1.4% Estimated total shards: ~1,000+ (accounting for failures)
