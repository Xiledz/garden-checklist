Suffix Pattern

The base suffixes cycle in groups of 10: K, M, B, T, Qd, Qn, Sx, Sp, Oc, No (representing 103 through 1030). After that, compound suffixes form by combining a base prefix with a group name: De (Decillion, 1033), Vt (Vigintillion, 1063), Tg (Trigintillion, 1093), qg (Quadragintillion, 10123), Qg (Quinquagintillion, 10153), sg (Sexagintillion, 10183), Sg (Septuagintillion, 10213), Og (Octogintillion, 10243), Ng (Nonagintillion, 10273), Ce (Centillion, 10303). Suffixes are sequential (+3 each), not multiplicative: SpDe = 1054, not 1057.

HOW IT WORKS

RPS = Bulk / Speed Prob = (1/Chance x Luck) / SourcePoolWeight ETA = 1 / (Prob x RPS) Runes/Hr = Prob x RPS x 3600 x (1 + Clone% x Clone#) Each rune button is a weighted pool. Weight per rune = min(1, Luck / Rarity). Pool total = sum of all weights on that button. Share = your weight / pool total.
